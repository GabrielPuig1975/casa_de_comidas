========================================
🍽️  CASA DE COMIDAS
Sistema de Gestion para Restaurantes
========================================

📌 REQUISITOS PREVIOS (SOLO UNA VEZ):

1. Instalar Node.js:
   - Ve a: https://nodejs.org/
   - Descarga la version LTS
   - Ejecuta el instalador (siguiente, siguiente, siguiente)

2. Instalar MySQL:
   - Ve a: https://dev.mysql.com/downloads/installer/
   - Descarga e instala
   - Cuando pida contrasena de root, escribe: root123
   - RECUERDA ESTA CONTRASENA

========================================

🚀 PRIMERA VEZ QUE USAS EL SISTEMA:

1. Haz doble clic en "INSTALAR.bat"
2. Espera a que termine (puede tomar varios minutos)
3. Cuando pida usuario y contrasena de MySQL:
   - Usuario: root
   - Contrasena: xxx (la que pusiste al instalar MySQL)

========================================

🚀 CADA VEZ QUE QUIERAS USAR EL SISTEMA:

1. Haz doble clic en "INICIAR.bat"
2. Se abrira el navegador automaticamente

========================================

📝 ACCEDER AL PANEL DE ADMINISTRACION:

- Ve a: http://localhost:5173/admin
- Usuario: admin
- Contrasena: admin123

========================================

❌ SOLUCION DE PROBLEMAS:

Si el sistema no inicia:
- Ejecuta "INSTALAR.bat" nuevamente

Si no reconoce "pnpm":
- Abre CMD como administrador
- Escribe: npm install -g pnpm
